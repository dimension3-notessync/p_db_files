# p_db_files

**Description:**  
database component only accessable via docker network.  
Handles read/write action requested by backend components on the 'files' collection
file entries are automatically assigned an unique (augmenting) id  


**API-rules**
```
Route               | HTTP-Method         | Input-Information                                                  | Output-Information              | Description
____________________________________________________________________________________________________________________________________________________________________________________
/                   | POST                | lectureID: Integer                                                 | 200 - message: String
                                          | authorID: Integer                                                  | id: Integer
                                          | fileName: String
                                          | fileUrl: String
                                          | subject: String
                                          | uploadTime: Integer (date)
                                          | (opt)aiSummary: String
____________________________________________________________________________________________________________________________________________________________________________________
/change/summary     | PUT                 | fileID: Integer                                                    | 200 - message: String
                                          | aiSummary: String
____________________________________________________________________________________________________________________________________________________________________________________
/:fileid            | DELETE              | fileid: Integer                                                    | 200 - message: String
____________________________________________________________________________________________________________________________________________________________________________________
/all                | GET                 | /                                                                  | 200 - files: Json[{}]
____________________________________________________________________________________________________________________________________________________________________________________
/author/:id         | GET                 | userid: Integer                                                    | 200 - files: Json[{}]                     
____________________________________________________________________________________________________________________________________________________________________________________
/subject/:id        | GET                 | id: Integer                                                        | 200 - files: Json[{}]                     
____________________________________________________________________________________________________________________________________________________________________________________
/lecture/:id        | GET                 | id: Integer                                                        | 200 - files: Json[{}] 
____________________________________________________________________________________________________________________________________________________________________________________
/:id                | GET                 | id: Integer                                                        | 200 - file: Json{}
____________________________________________________________________________________________________________________________________________________________________________________
/health             | GET                 | /                                                                  | 200 - message: String             | Added for docker-health checks
                                        
```
